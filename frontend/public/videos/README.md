# Ending Videos Folder

Place your 4 custom ending MP4 videos inside this folder. They will immediately autoplay and loop on the final ending screens!

## Required Filenames:
1. `birthday-ending.mp4` (Best Birthday Ending)
2. `musical-ending.mp4` (Chaotic Musical Ending)
3. `ghost-ending.mp4` (Ghost of the Gielgud Standing Ovation)
4. `cake-ending.mp4` (Great Cake Incident Ending)

*Note: For the best visual result, use standard landscape-oriented MP4 video files.*
